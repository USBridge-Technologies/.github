## What we build

### 🖥️ USBridge KVM 2.0
<div align="center">
<img src="https://raw.githubusercontent.com/USBridge-Technologies/.github/main/profile/KVM_USBridge.png" width="100%" />
 </div>
 
Hardware KVM-over-IP via a single USB connection. BIOS access over SSH, no agents, no cloud.

- BIOS-in-Terminal — OCR on-device, text stream over SSH
- Immutable Btrfs snapshots — host can't touch them
- Virtual media & ISO mounting
- ATX power control

→ [Pre-launch on Crowdsupply](https://crowdsupply.com/usbridge-technologies/usbridge-kvm-2-0)

---

### 💻 USBridge Remote Access
Free remote desktop software. Works standalone or with KVM hardware.

- Built-in Tailscale — zero config, no port forwarding
- Agent + Client for Windows, macOS, Linux
- Software mode or hardware-augmented mode

→ [USBridge-Remote](https://github.com/USBridge-Technologies/USBridge-Remote)
